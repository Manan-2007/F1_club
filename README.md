# F1 Chitkara

Official website of the F1 Chitkara club — Chitkara University.
Where Speed Meets Innovation.

## Stack
Vite · React · Tailwind CSS · GSAP · Framer Motion · React Three Fiber · Lenis

## Setup
1. Clone the repo
2. `cp .env.example .env.local` and fill in Firebase credentials
3. `npm install`
4. `npm run dev`

## Deploy
Configured for Vercel. Connect repo on vercel.com.
Add `.env.local` values as Vercel environment variables.
